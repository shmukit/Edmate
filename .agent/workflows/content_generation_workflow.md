---
description: Workflow for generating and formatting modular educational content using Gemini and ChatGPT
---

This workflow outlines the steps to generate curriculum-agnostic educational content. While the examples below use A/O-Level subjects, the prompts can be adjusted for any subject or standard.

## Step 1: Prepare Source Materials
Gather the source materials (e.g., Question Papers, Study Guides, or Transcripts) and any reference keys (Marks Schemes).

## Step 2: Generate Content with Gemini
Use **Gemini** to generate detailed explanations, concept gap analysis, and flashcards.

**Prompt for Gemini:**
> [!NOTE]
> Replace `[Question Paper Name]` and `[Marks Scheme Name]` with actual filenames.
> Adjust the question range `1-10` as needed.

```text
[Question Paper Name] is the question paper and [Marks Scheme Name] is the answer of questions (1-40).

For the provided Biology questions 1-10, please generate a detailed analysis using the following structure and formatting rules:

Write the question number

Question and Options in Text Format Provide the question text and all options.

Detailed Explanation of the Question and Right Answer Provide a short explanation of the overall method before the step-by-step analysis. Use the following step-by-step analysis structure for the main explanation: State the core concept.

Analyze Step 1, concluding the intermediate result.

Analyze Step 2, concluding the intermediate result.

Analyze Step 3, detailing the calculation/reasoning, and concluding the result.

Analyze Step 4, detailing the calculation/reasoning, and concluding the result (use only if necessary).

State the final correct answer based on the analysis.

Option Wise Explanation (Detailed) Provide a detailed explanation for why each option (A, B, C, D) is correct or incorrect. Present the explanation in paragraph format (no tables).

Each Option Wise Concept Gap Analysis and for each option separate 2-3 Flashcards.

Start this section with the header:

### 🧠 Concept Gap Analysis and Flashcards. For every individual wrong option (e.g.,A, B, C, D), identify the conceptual gap that led to that specific incorrect choice.

### Following the gap analysis, provide 2-3 separate tailored flashcards for the each options.

Format the flashcards in text/paragraph format for each.

Start each new flashcard on a new line.

Format flashcards as: Flashcard X: [Front text]? Back: [Back text].

General Formatting and Content Rules (Strictly Follow):

Use standard markdown headings (### and ##) to create a clear hierarchy.

Ensure all content is presented in paragraph format (no tables, no UI elements).

Be detailed and rigorous in all explanations, adhering to the specified subject syllabus standards (e.g., 9700 Biology AS).

The final output must only contain the requested sections in order.

Ensure the short explanation in section 2 is brief and precedes the step-by-step analysis.
```

## Step 3: Format Content with ChatGPT
Use **ChatGPT** to convert the Gemini output into a format compatible with Google Docs (Unicode subscripts, emoji preservation, etc.).

**Prompt for ChatGPT:**
> [!NOTE]
> Paste the content generated by Gemini after the prompt below.

```text
Rewrite the following document for Google Docs compatibility: - Convert all LaTeX math (like $E_a$, $e^{-E_a/RT}$, $\frac{3}{2}kT$) into readable text form using Unicode subscripts/superscripts (e.g., Eₐ, e^(–Eₐ/RT), ³⁄₂kT). - Preserve Greek symbols (Δ, α, β) in Unicode. - Keep all emoji (🅰️, 🅱️, 🧠) and section headers intact. - Maintain indentation, lists, and step numbering. - Remove dollar signs or LaTeX markup while keeping equations visually clear. - Output as plain text for direct pasting into Google Docs without breaking. Do not reduce text, keep it as it is.

[PASTE GEMINI OUTPUT HERE]
```

## Step 4: Store in Word File
Copy the output from ChatGPT and paste it into a Microsoft Word file.

## Step 5: Upload to System
Paste the content from the Word file into the system's upload interface.
