# Edmate Brand Guideline

This document defines the visual identity for the Edmate Automation Hub, derived from the core favicon design. The aesthetic focuses on technical precision, minimalism, and soothing high contrast.

## 🎨 Color Palette

### Primary Colors
| Color | Base Hex | Usage |
| :--- | :--- | :--- |
| **Edmate Night** | `#111827` | Primary background for dark mode, sidebar headers. |
| **Edmate Deep Blue**| `#1e1b4b` | Secondary backgrounds, cards, and deep accents. |
| **Edmate Amber** | `#fbbf24` | Primary brand color, primary buttons, lightning bolt icons. |
| **Edmate Glow** | `#fff3b0` | Subtitle highlights, secondary yellow accents. |

### Functional Neutrals
- **Dark Background**: `#111827` (Night Surface)
- **Light Background**: `#f8fafc` (Slate Light)
- **Text Main (Dark)**: `#f1f5f9`
- **Text Main (Light)**: `#0f172a`

## 📐 Design Philosophy

### 1. Minimalist Character
- Avoid "AI-gradient" overkill. Use solid colors.
- Icons should be single-colored (Amber or White/Dark Blue depending on theme).
- Focus on spacing and typography over decorative elements.

### 2. Subtle Textures (Grid/Mesh)
- **Dark Mode**: A subtle `#fbbf24` opacity (4-5%) grid pattern over the dark background.
- **Light Mode**: A subtle `#fbbf24` opacity (10-12%) grid pattern over the light background.
- Grid size: 32px to maintain a technical "blueprint" feel.

### 3. Typography
- **Primary**: `Inter` (Sans-serif) for body and UI elements.
- **Header**: `Outfit` (Sans-serif) for titles to provide a modern, friendly character.

## 🌗 Theming Strategy

- **Dark Theme**: "Midnight Technical". High contrast between the deep navy and bright amber.
- **Light Theme**: "Clean Studio". A breathable light interface with deep yellow accents for warmth without being overwhelming.
